# تثبيت قاعدة البيانات الجديدة الضخمة

## اسم قاعدة البيانات الجديدة:
**`arhgo_new_db`**

---

## خطوات التثبيت:

### 1. افتح phpMyAdmin:
```
http://localhost/phpmyadmin
```

### 2. نفذ الملفات بالترتيب التالي:

#### أ) إنشاء قاعدة البيانات والجداول:
1. اضغط على تبويب **SQL**
2. **امسح أي نص موجود في المربع**
3. انسخ محتوى `database/CREATE_NEW_DATABASE.sql` بالكامل
4. الصقه في المربع
5. اضغط **Go** (تنفيذ)

#### ب) إدراج الدول:
1. في نفس phpMyAdmin، اضغط **SQL**
2. **امسح أي نص موجود**
3. انسخ محتوى `database/INSERT_COUNTRIES_NEW.sql` بالكامل
4. الصقه واضغط **Go**

#### ج) إدراج المدن:
1. اضغط **SQL**
2. **امسح أي نص موجود**
3. انسخ محتوى `database/INSERT_CITIES_NEW.sql` بالكامل
4. الصقه واضغط **Go**

#### د) إدراج الفنادق:
1. اضغط **SQL**
2. **امسح أي نص موجود**
3. انسخ محتوى `database/INSERT_HOTELS_NEW.sql` بالكامل
4. الصقه واضغط **Go**

#### هـ) إدراج المطاعم:
1. اضغط **SQL**
2. **امسح أي نص موجود**
3. انسخ محتوى `database/INSERT_RESTAURANTS_NEW.sql` بالكامل
4. الصقه واضغط **Go**

---

## ملاحظات مهمة:

⚠️ **تأكد من:**
- مسح أي نص موجود في مربع SQL قبل اللصق
- نسخ الملف بالكامل من السطر الأول إلى الأخير
- تنفيذ الملفات بالترتيب المذكور أعلاه

---

## التحقق من التثبيت:

بعد تنفيذ جميع الملفات، افتح:
```
http://localhost/ArhGo-trip-planner/api/check_db.php
```

يجب أن ترى:
- ✅ عدد الدول: 12
- ✅ عدد المدن: 70+
- ✅ عدد الفنادق: 200+
- ✅ عدد المطاعم: 150+

---

## تحديث ملف config.php:

تم تحديث `api/config.php` لاستخدام قاعدة البيانات الجديدة:
```php
define('DB_NAME', 'arhgo_new_db');
```
